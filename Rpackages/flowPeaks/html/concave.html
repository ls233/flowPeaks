concave {flowPeaks}R Documentation

The concave dataset

Description

A simulated flowcytometry data with two concave shapes

Usage

data(concave)

Format

An object (concave) of data frame with rows and 3 columns and a vector (concave.cid) for the true cluster labels.

References

Ge Y. et al, flowPeaks: a fast unsupervised clustering for flow cytometry data via K-means and density peak finding, 2012, Bioinformatics, in press.
